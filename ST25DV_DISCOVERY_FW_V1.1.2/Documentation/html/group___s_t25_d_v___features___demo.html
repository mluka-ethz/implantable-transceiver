Detailed Description

This module implements the functions to execute the demos of the ST25DV specific features.

This module covers the following usecases:

  • Energy Harvesting
  • GPO interrupts from RF
  • ST25DV states: RF off, RF sleep, Low power down
  • Memory map configuration and Password protection
