Binary Installation

ST-Link debugger is embedded on the ST25-Discovery Board. When connecting a mini USB cable to the board from your computer, this will automatically add a new drive in your explorer window (Windows only). Drag and Drop the binary file on the new drive added in the explorer view (Windows). After file succeed to be copied, reset the Discovery board.
Or you can use your favorite compiler tools to flash the binary on the ST25-Discovery board.
