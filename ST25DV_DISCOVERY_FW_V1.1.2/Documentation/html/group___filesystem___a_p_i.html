Detailed Description

Simple implementation of a filesystem API.

This module implements the functions to open, read and close a file. It does NOT implements a real filesystem. A file is considered being a starting address in memory, from where the API reads a stream of data. The purpose of this module is to provide a standard filesystem api to feed data to the libJPEG.

FRESULT f_close ( FIL *  fp)

Close a stream of data from an address in memory.

This function mimic the close standard function from a filesystem. The usage of such function in the ST25DV demo is to feed data to the jpeg decoder module, without having a real filesystem.

Parameters
fpPointer to FIL structure - mimic a filehandle.
Return values
FR_OKData stream has been successfully closed.
FR_INVALID_OBJECTfp is not allocated or not open.
